# 幼儿园老师 Agent 产品需求文档（PRD）

## 1. 产品概述

### 1.1 产品定位

面向幼儿园教师的专属 AI 文案生成 Agent 网站，通过专业化 Prompt 工程、幼教领域知识注入和 RAG 检索增强，帮助教师高效完成教育教学、行政运营、家长沟通等多类文案创作。解决通用大模型在幼教场景下内容模版化、缺乏循序渐进、创新不足、专业性不够及幻觉等问题。

### 1.2 用户画像

| 维度 | 描述 |
|------|------|
| 角色 | 幼儿园教师（主班、配班、保育员、教研组长等） |
| 工作特点 | 事务繁杂，文案需求多样，时间碎片化 |
| 痛点 | 重复性文案多、专业要求高、希望内容有特色、需符合园所风格 |
| 技术能力 | 中等，能使用 Web 应用，偏好简单直观的界面 |

### 1.3 核心价值

- 专业：基于幼教领域知识库，减少幻觉与不专业表述
- 个性化：支持上传个人资料供 AI 学习，输出更贴合园所风格
- 高效：一站式生成多类文案，减少重复劳动
- 可控：可按年龄段、风格、字数等参数定制输出

## 3. 核心交互

### 3.1 主工作台

- 左侧或顶部：文案类型分类选择（树形/卡片）
- 中央：聊天式输入框 + 对话历史
- 右侧或折叠面板：参数配置区
  - 年龄段：托班 / 小班 / 中班 / 大班 / 混龄
  - 风格：活泼 / 严谨 / 温馨 / 专业 / 简洁
  - 字数：短（约 300 字）/ 中（约 600 字）/ 长（约 1000 字）/ 自定义
  - 是否引用个人知识库：开关

### 3.2 聊天流程

1. 用户选择文案类型
2. 系统自动填充默认参数，用户可调整
3. 用户在输入框描述需求（可简要）
4. 系统组装 Prompt，调用大模型生成
5. 流式展示生成结果，支持复制、编辑、重新生成
6. 引用来源以脚注或侧边栏展示（如有）

### 3.3 创意启发模块（可选增强）

- 输入主题或关键词后，展示「活动形式建议」「素材联想」等
- 支持用户选择后再进入正式生成

## 4. 用户系统

### 4.1 登录与注册

- 支持手机号 + 验证码 或 邮箱 + 密码 注册/登录
- 支持第三方登录（微信、钉钉等，视集成能力）
- 未登录用户可有限试用（如每日 3 次生成），需登录后解锁完整能力

### 4.2 订阅

- 免费版：每日有限次数，基础文案类型
- 付费版：更高次数或无限，全部文案类型，个人知识库容量更大
- 订阅管理：升级、续费、取消、发票

### 4.3 个人知识库

- 支持上传 PDF、Word、TXT 等文件
- 文件经解析、分块、向量化后存入私有知识库
- 生成时可选「使用我的知识库」进行 RAG 检索
- 支持查看、删除已上传文件，查看使用统计（如检索次数）

## 5. 信息架构与关键流程

### 5.1 信息架构

```
首页/工作台
├── 文案生成（主功能）
│   ├── 教育教学与教研
│   ├── 班级管理与环创
│   ├── 家长工作与活动
│   ├── 卫生保健与安全
│   └── 行政宣传
├── 我的知识库
│   ├── 上传文件
│   └── 文件列表与管理
├── 订阅中心
│   ├── 套餐选择
│   └── 订单/发票
└── 个人中心
    ├── 账号设置
    ├── 使用统计
    └── 退出登录
```

### 5.2 关键用户流程

**流程一：生成文案（已登录）**

```
选择文案类型 → 填写/调整参数 → 输入需求描述 → 生成 → 查看/复制/编辑
```

**流程二：上传知识库**

```
我的知识库 → 上传文件 → 等待解析完成 → 生成时勾选「使用我的知识库」
```

**流程三：订阅升级**

```
订阅中心 → 选择套餐 → 支付 → 订阅生效 → 解锁权益
```

## 6. 非功能需求

### 6.1 性能

| 指标 | 要求 |
|------|------|
| 首屏加载 | ≤ 3 秒（普通网络） |
| 生成首字响应 | ≤ 5 秒（流式） |
| 单次生成完整响应 | 视字数，一般 ≤ 30 秒 |
| 并发 | 支持 100+ 在线用户同时生成（可随业务扩展） |

### 6.2 安全与隐私

- 用户数据加密存储与传输（HTTPS、敏感字段加密）
- 个人上传文件仅本人可访问，不用于训练公开模型
- 登录态与 Token 安全管理，支持登出与异地登录提醒

### 6.3 可用性

- 支持主流浏览器（Chrome、Safari、Edge、Firefox）
- 移动端适配（响应式），便于教师手机使用
- 关键操作有明确反馈，错误提示友好可操作

### 6.4 可维护性

- 文案类型、Prompt 模板支持后台配置与迭代
- 日志与监控完备，便于排查问题与优化效果
